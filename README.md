# Image uploader

## install packages

```shell
npm/yarn/pnpm install
```

## Run app

```shell
npm/yarn/pnpm dev
```

- Server should be running at `http://localhost:3000`
- backend server should be running on `http//localhost:5005`, [which can be found here.](https://github.com/phacic/gecko-images-backend)
- Supports dark mode
